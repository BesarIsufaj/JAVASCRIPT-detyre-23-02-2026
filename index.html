Ushtrimi 1: Ndryshimi i Tekstit me Buton

Krijo një faqe HTML me një paragraf dhe një buton. Kur përdoruesi klikon butonin, teksti i paragrafit duhet të ndryshojë në: "JavaScript po funksionon!"

▸ addEventListener ▸ innerHTML / textContent ▸ HTML bazë

Kodi


                
<p id="paragraf1">Shtypni butonin për të ndryshuar tekstin</p>
<button id="btn-ndryshoText">Butoni</button>

<script>
  let paragrafi = document.getElementById("paragraf1")
  let btn = document.getElementById("btn-ndryshoText")
  
  btn.addEventListener("click", () => {
  paragrafi.textContent = "JavaScript po funksionon!";

</script>

Ushtrimi 2: Mbledhja e Dy Numrave

Krijo dy fusha input për numra dhe një buton Llogarit. Kur klikohet butoni, programi duhet të shfaqë shumën e dy numrave.

▸ input type='number'

Kodi


<label>Numër 1:<input type="number" name="numer1" id="numer1"/></label></br>
<label>Numër 2: <input type="number" name="numer2" id="numer2"/></label></br>
<button onclick="gjejShumen()">Llogarit shumën</button>
<p id="shuma"></p>

<script>
let nr1 = document.getElementById("numer1");
let nr2 = document.getElementById("numer2");
let paragraf = document.getElementById("shuma");

function gjejShumen() {
  let rezultati = isNaN(nr1.valueAsNumber) ? 0
   : nr1.valueAsNumber;
  rezultati += isNaN(nr2.valueAsNumber) ? 0
   : nr2.valueAsNumber;

  paragraf.textContent = "Shuma është: " + rezultati;
}
</script>

Output

Ushtrimi 4: Çift apo Tek?

Kodi

<input type="number" name="numer4" id="numer4">
<p id="paragraf4"></p> 

<script>
let input4 = document.getElementById("numer4");
let p = document.getElementById("paragraf4");

function check() {
  console.log("enter function");
  let mesazhi = "";
  let numer = input4.valueAsNumber;
  if (numer % 2 == 0) {
    mesazhi = "Numri " + numer + " është ÇIFT";
  } else {
    mesazhi = "Numri " + numer + " është TEK";
  }
  p.textContent = mesazhi;
}
</script>

Ushtrimi 6: Lista Dinamike

Krijo një fushë input dhe një buton Shto. Teksti i shkruar në input duhet të shtohet si element i ri në një listë (ul).

createElement('li') ▸ appendChild() ▸ Pastrim input

Kodi


<h3>Shto elementë në listë</h3>
<input type="text" name="input6" id="input6">   
<button onclick="shtoItem()">Shto në listë </button>
<ul id="list6"><b>Lista</b></ul>

<script>
let itemText = document.getElementById("input6");
let lista = document.getElementById("list6");

function shtoItem() {
  let item = document.createElement('li');
  item.textContent = itemText.value;
  lista.appendChild(item);
  itemText.value = "";
}
</script>

Output

Shto elementë në listë

    Lista

Validimi i Fjalëkalimit

Krijo një fushë për fjalëkalim. Nëse fjalëkalimi ka më pak se 6 karaktere, shfaq mesazhin: "Fjalëkalimi është shumë i shkurtër" — përndryshe: "Fjalëkalim i vlefshëm"

▸ input type='password' ▸ Prona .length ▸ if / else

Kodi


<label>Pasword: <input type="password" name="psw7" id="psw7" /></label>
<p id="mesazh7"></p>
<script>
let input7 = document.getElementById("psw7");
let mesazh = document.getElementById("mesazh7");

input7.addEventListener("input", function () {
  if (input7.value.length < 6) {
    mesazh.textContent = "Fjalëkalimi është shumë i shkurtër";
    mesazh.style.color = "red";
  } else {
    mesazh.textContent = "Fjalëkalim i vlefshëm";
    mesazh.style.color = "green";
  }
});
</script>

Ushtrimi 8: Numri Më i Madh

Merr 3 numra nga përdoruesi dhe shfaq në faqe numrin më të madh.

▸ Math.max() ▸ if / else if ▸ Tre inpute

Kodi


<h3>Jepni 3 numra dhe gjej më të madhin</h3>
<label>Numri 1: <input type="number" id="num1" /></label><br />
<label>Numri 2: <input type="number" id="num2" /></label><br />
<label>Numri 3: <input type="number" id="num3" /></label><br />
<button onclick="gjejMax()">Gjej Numrin më të Madh</button>


<script>
function gjejMax() {
  let n1 = parseFloat(document.getElementById("num1").value) || 0;
  let n2 = parseFloat(document.getElementById("num2").value) || 0;
  let n3 = parseFloat(document.getElementById("num3").value) || 0;

  let maxNum = Math.max(n1, n2, n3);
  document.getElementById("rezultati").textContent =
    "Numri më i madh: " + maxNum;
}
</script>

Output

Jepni 3 numra dhe gjej më të madhin
